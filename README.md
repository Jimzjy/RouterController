# RouterController

一个路由管理 App , 通过 SSH 连接路由器.

![license](/tmp/license-Apache_2.0-blue.svg) ![api](/tmp/API-21+-blue.svg)

功能 :

- 显示网速
- 显示连接设备
- 发送命令
- 更新系统时间
- 修改设置
- 上传文件
- 闪讯密码更新
